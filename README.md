# Home Assistant Blueprints

A collection of Home Assistant automation blueprints.

**[HUE Smarter Button (Multi-press)](Philips_HUE_SmartButton.yaml)**
Multi-press controller for Philips HUE Smart Button. Supports up to 5 press patterns, hold-to-dim, and Hue/ZHA/Zigbee2MQTT integrations.

**[Renew Let's Encrypt Certificate](renew_ssl_cert.yaml)**
Automatically renews your Let's Encrypt certificate when expiry is near.
